# Karlytta / Prudina Archive Package

Open `index.html` in a browser. The site is fully static and includes compressed local audio.

## Included

- Working archive-site prototype
- Original and extended playable audio in OGG
- Cover artwork
- Objective audio metrics and waveform images
- Canon bible
- Lawsuit architecture
- Breaking-news article
- Narrative/worldbuilding production system
- Research basis
- 148-item concrete asset backlog
- Personas, claims, timeline and forum data in JSON
- SQLite database
- CSV analysis files

## Important status labels

- **confirmed**: established inside current canon
- **alleged**: asserted by a named source
- **disputed**: materially present but authenticity or meaning contested
- **unknown**: insufficient basis
- **performative**: true as an act or public position, not necessarily as factual description
