# Slegnuće Narrative Production System

## The separation that protects the game

The website is Prudina's memory.  
The game is the Doctor's experience.

The website may be broader, louder and more encyclopedic. The game must remain physically immediate and emotionally selective. A player who never opens the site must still understand the Doctor’s central conflict. A reader who never plays the game should still understand how Prudina manufactures public reality.

## The three-ledger method

Maintain three separate ledgers.

### 1. Game ledger
Only scenes, mechanics, locations, decisions and information required for the playable story.

### 2. World ledger
Institutions, wars, public figures, media systems, laws, geography, cultural memory.

### 3. Artifact ledger
Concrete things that can be produced: pages, recordings, tickets, forms, images, transcripts, databases.

Never move an item into the game ledger merely because it is interesting. It enters only when it changes player action, emotional interpretation or spatial design.

## Weekly production ratio

For a solo project:

- 70% playable vertical slice;
- 20% world-site artifacts that reuse existing game material;
- 10% free experimentation.

Karlytta work counts as game work only when it creates a reusable prop, sound, television segment, legal form, UI system, location texture or choice.

## The return test

Before producing an artifact, answer:

1. What pressure does it reveal?
2. What existing asset does it reuse?
3. Which character becomes more legible because of it?
4. Can it be completed in one session?
5. Does it create a link rather than a new branch?
6. What is the cheapest in-game echo?
7. What would be lost if it did not exist?

If the final answer is “nothing except lore volume,” place it in cold storage.

## Artifact dramaturgy

A strong artifact contains four simultaneous stories:

1. **Declared purpose** — what the document says it is for.
2. **Operational purpose** — what it actually makes someone do.
3. **Human residue** — handwriting, delay, stain, correction, omission, compression artifact.
4. **Political consequence** — who gains control of time, movement, language or credibility.

Example:

A clinic visitor form declares that it protects patients. Operationally, it allows a media official to enter under “security.” A nurse has crossed out the title and written a name. The political consequence is that celebrity privacy becomes administratively optional.

## Scale ladder

Every major theme should appear at several scales.

| Scale | Example |
|---|---|
| Body | voice strain, medication, touch, refusal |
| Room | locked corridor, camera angle, missing chair |
| Building | access control, maintenance, heat, queues |
| City | transport, closures, rumours, screens |
| Institution | legal categories, archive ownership |
| Country | continuity, symbolism, emergency rhetoric |
| History | war memory, inherited silence, recurring procedures |

Do not write a speech about authoritarianism when a door schedule can perform it.

## Conflict matrix

For every major character, define conflict on five levels:

- **Internal:** what they cannot admit about themselves.
- **Interpersonal:** who can force them to become a version they hate.
- **Institutional:** which procedure rewards their flaw.
- **Material:** what resource is running out.
- **Historical:** which inherited story determines what seems possible.

Karlytta:
- internal: cannot separate self-expression from survival performance;
- interpersonal: Varda makes refusal feel childish;
- institutional: RP Network rewards volatility and punishes privacy;
- material: access to broadcast, money, staff loyalty and health;
- historical: public women become symbols before they are allowed to be witnesses.

The Doctor:
- internal: confuses mastery with care;
- interpersonal: family needs presence, not diagnosis;
- institutional: medicine rewards distance until distance becomes neglect;
- material: time, heat, medicine, transport, sleep;
- historical: survival culture treats vulnerability as betrayal.

## The contradiction budget

Each article should contain:
- one fact that appears stable;
- one source that complicates it;
- one material detail that cannot be explained away;
- one interpretation that is emotionally convincing but evidentially weak;
- one question that becomes sharper rather than solved.

Do not make every item uncertain. Total ambiguity is merely fog with a database.

## Loud voices without caricature

The loudest public voices should possess:
- a real area of competence;
- a private dependency;
- one accurate observation;
- one recurring evasion;
- one phrase that followers repeat;
- one person they cannot dominate;
- and one mundane task they still have to perform.

This prevents politicians, hosts, clerics, critics and celebrities from becoming theme-delivery machines.

## Site architecture

The production-ready content model is:

- `people`
- `institutions`
- `events`
- `articles`
- `comments`
- `tracks`
- `artifacts`
- `claims`
- `evidence_links`
- `relationships`
- `locations`
- `editions`
- `moderation_actions`
- `access_logs`

Every public page is assembled from these records. Do not hard-code lore into layout after the prototype stage.

## One-month vertical slice

### Week 1: Lawsuit ignition
- Publish the breaking article.
- Add official and extended audio.
- Add eight comments with incompatible readings.
- Create one legal cover sheet and one RP correction notice.
- Add one waiting-room television screen to the game.

### Week 2: Evidence and routine
- Add claim statuses and evidence links.
- Build one clinic access-log prop.
- Create a corridor scene where the Doctor decides whether to report improper access.
- No new Karlytta biography.

### Week 3: Public spectacle
- Build one NARODNA KUĆA special page.
- Create host cue cards, voting UI and moderation notices.
- Reuse the same broadcast in two in-game locations with different sound conditions.

### Week 4: Consequence
- Add one witness who loses something practical.
- Reflect the lawsuit in transport, clinic behaviour or family conversation.
- Test whether removing every Karlytta reference still leaves the Doctor’s scene dramatically functional.
- Freeze the archive and return to the main vertical slice.

## Definition of done for a Karlytta article

An article is done when:
- it has a dateline and accountable publisher;
- it distinguishes allegation from established fact;
- its bias is visible through selection rather than cartoon lying;
- it links one track, one artifact and one prior event;
- comments contain at least three genuinely different epistemologies;
- it changes one relationship in the database;
- it can be read in under seven minutes;
- and it does not require another article to become meaningful.
